Vytvořte pomocí PHP a javascriptu/jQuery stránku s jednoduchým formulářem k odeslání, kde si kupující vyplní základní osobní informace a bude moci zadat produkt včetně ceny.
Formuláři dopřejte pěkný vzhled pomocí CSS.
Nezapomeňte, že změnou počtu kusů se musí změnit i celková cena.
Nezapomeňte na validaci jednotlivých prvků formuláře.
Na ní se po odeslání formuláře zobrazí rekapitulace včetně DPH.
Rekapitulace bude obsahovat přepočet ceny do libovolné jiné měny dle aktuálního kurzovního lístku České národní banky, který se automaticky načte.
